📘 Nível 1.3 — Medidas, Unidades e Erros

🔹 Medir é comparar

Medir uma grandeza significa compará-la com um padrão.

Quando dizemos que algo mede 2 metros, estamos comparando esse comprimento com o padrão chamado metro.

🔹 Unidades de medida

As unidades são os padrões usados para medir.

  • Comprimento → metro (m)
  • Tempo → segundo (s)
  • Massa → quilograma (kg)
  • Temperatura → grau Celsius (°C)

Usar unidades padronizadas permite que pessoas diferentes compreendam a mesma medida.

🔹 Medidas não são perfeitas

Nenhuma medida é absolutamente exata. Toda medida possui um erro de medida.

O erro não significa que a medida está errada, mas que existe uma aproximação.

🔹 Exemplos de erro de medida

  • Duas pessoas medem o mesmo objeto e obtêm valores ligeiramente diferentes
  • Um cronômetro é acionado com atraso
  • Um instrumento não é perfeitamente calibrado

🔹 Por que entender erros é importante?

Entender erros de medida ajuda a:

  • Interpretar resultados corretamente
  • Evitar conclusões erradas
  • Comparar medições

🧠 Resumo do Nível

  • Medir é comparar com um padrão
  • Unidades padronizam as medições
  • Toda medida possui erro
  • Erro não significa medida errada
