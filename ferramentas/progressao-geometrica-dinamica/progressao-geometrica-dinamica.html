Fórmulas importantes da Progressão Geométrica

Utilize esta área como consulta rápida durante a aula.

Termo geral
aₙ = a₁ · q^(n - 1)

Permite descobrir qualquer termo da progressão.

Razão
q = aₙ / aₙ₋₁

A razão é o número pelo qual cada termo é multiplicado.

Soma dos termos
Sₙ = a₁(qⁿ - 1) / (q - 1)

Fórmula usada para calcular rapidamente a soma da PG.

Quando q = 1
Sₙ = a₁ · n

Caso especial em que todos os termos da PG são iguais.

Último termo
aₙ = a₁ · q^(n - 1)

Encontra diretamente o último termo da sequência.

Desafio rápido
Se a₁ = 3 e q = 2, qual é o 6º termo?
