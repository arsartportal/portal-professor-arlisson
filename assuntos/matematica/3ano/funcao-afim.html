🎯 Compreender a estrutura da função afim, interpretar seu gráfico e aplicar em situações reais.

1️⃣ Definição

A função afim é definida por:

f(x) = ax + b

Onde: ✔ a = coeficiente angular (inclinação da reta) ✔ b = coeficiente linear (intercepto no eixo y)

2️⃣ Interpretação do Coeficiente Angular (a)

✔ a > 0 → função crescente ✔ a < 0 → função decrescente ✔ a = 0 → função constante

3️⃣ Interpretação do Coeficiente Linear (b)

É o valor onde a reta corta o eixo y (quando x=0).

4️⃣ Zero da Função

O zero ocorre quando f(x)=0.

ax + b = 0 → x = -b/a

5️⃣ Gráfico

O gráfico da função afim é sempre uma reta.

6️⃣ Aplicações Reais

✔ Cálculo de salário fixo + comissão ✔ Tarifa de táxi (bandeirada + valor por km) ✔ Conversão de temperatura ✔ Custo fixo + custo variável

7️⃣ Exemplo Resolvido

Uma corrida de aplicativo cobra R$5 de taxa fixa e R$2 por km.

f(x)=2x+5 Para 10 km: f(10)=25 reais.

8️⃣ Erros Comuns

❌ Confundir coeficiente angular com linear ❌ Esquecer o sinal negativo ❌ Resolver errado o zero da função

9️⃣ Comparação com Função Constante

Se a=0, temos f(x)=b (reta horizontal).

🔟 Conexão

A função afim é base para sistemas lineares, inequações e geometria analítica.
