1️⃣ Definição

Uma função exponencial é definida por:

f(x) = ax

Onde: ✔ a > 0 ✔ a ≠ 1 ✔ x está no expoente

2️⃣ Crescimento e Decrescimento

✔ Se a > 1 → função crescente ✔ Se 0 < a < 1 → função decrescente

3️⃣ Propriedades das Potências

ax+y = ax · ay (ax)y = axy a−x = 1 / ax

4️⃣ Interpretação Gráfica

✔ O gráfico nunca toca o eixo x. ✔ Sempre passa pelo ponto (0,1). ✔ Tem assíntota horizontal em y=0.

5️⃣ Equações Exponenciais

Exemplo:

2x = 8 8 = 2³ → x = 3

6️⃣ Aplicações Reais

✔ Crescimento populacional ✔ Juros compostos ✔ Radioatividade ✔ Pandemias ✔ Tecnologia e dados digitais

7️⃣ Exemplo Aplicado

Uma população cresce 10% ao ano:

P(t)=P₀·(1,1)t

8️⃣ Comparação com Função Afim

Função afim cresce linearmente. Função exponencial cresce aceleradamente.

9️⃣ Erros Comuns

❌ Confundir base com expoente ❌ Esquecer que a ≠ 1 ❌ Errar propriedades de potência

🔟 Conexão

Base para função logarítmica.
