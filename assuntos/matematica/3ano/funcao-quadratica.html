🎯 Compreender a estrutura da parábola, suas raízes, vértice e aplicações.

1️⃣ Definição

A função quadrática é dada por:

f(x) = ax² + bx + c

Onde: ✔ a ≠ 0 ✔ a determina concavidade ✔ b e c influenciam posição da parábola

2️⃣ Concavidade

✔ a > 0 → concavidade para cima (mínimo) ✔ a < 0 → concavidade para baixo (máximo)

3️⃣ Discriminante (Δ)

Δ = b² − 4ac

✔ Δ > 0 → duas raízes reais ✔ Δ = 0 → uma raiz real ✔ Δ < 0 → não possui raízes reais

4️⃣ Fórmula de Bhaskara

x = (-b ± √Δ) / (2a)

5️⃣ Vértice

xv = -b / (2a) yv = -Δ / (4a)

O vértice representa o ponto de máximo ou mínimo da função.

6️⃣ Interpretação Gráfica

O gráfico da função quadrática é uma parábola.

7️⃣ Exemplo Resolvido

f(x)=x²−4x+3 Δ=16−12=4 x=1 e x=3 Vértice: x=2 f(2)= -1

8️⃣ Aplicações Reais

✔ Trajetória de projéteis ✔ Economia (lucro máximo) ✔ Engenharia ✔ Física (movimento uniformemente variado)

9️⃣ Erros Comuns

❌ Errar o sinal no cálculo do Δ ❌ Esquecer que a ≠ 0 ❌ Confundir raízes com vértice
