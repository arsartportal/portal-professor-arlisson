🎯 Compreender crescimento linear e exponencial por meio das progressões.

1️⃣ Progressão Aritmética (PA)

Uma PA é uma sequência onde cada termo é obtido somando uma constante r (razão).

an = a₁ + (n−1)r

2️⃣ Soma dos Termos da PA

Sn = n(a₁ + an) / 2

3️⃣ Aplicações da PA

✔ Parcelamentos ✔ Crescimento linear ✔ Economia ✔ Planejamento financeiro

4️⃣ Progressão Geométrica (PG)

Uma PG é uma sequência onde cada termo é obtido multiplicando por uma constante q.

an = a₁ · qn−1

5️⃣ Soma dos Termos da PG

Sn = a₁ (qn − 1) / (q − 1), q ≠ 1

6️⃣ PG Infinita

Se |q| < 1:

S = a₁ / (1 − q)

7️⃣ Exemplo PA

2, 5, 8, 11... r = 3

8️⃣ Exemplo PG

3, 6, 12, 24... q = 2

9️⃣ Comparação

PA → crescimento linear PG → crescimento exponencial

🔟 Aplicações Reais

✔ Juros compostos (PG) ✔ Depreciação ✔ Crescimento populacional ✔ Economia
