📘 Função Logarítmica

Matemática • 3º Ano • Professor: Arlisson Ferreira

🎯 Compreender logaritmos como inversos da função exponencial e aplicá-los em problemas reais.

1️⃣ Definição

Se ax = b, então:

loga b = x

Onde: ✔ a > 0 ✔ a ≠ 1 ✔ b > 0

2️⃣ Relação com Função Exponencial

A função logarítmica é a inversa da função exponencial.

f(x)=logax

3️⃣ Domínio

A função logarítmica só existe para x > 0.

4️⃣ Crescimento e Decrescimento

✔ Se a > 1 → função crescente ✔ Se 0 < a < 1 → função decrescente

5️⃣ Propriedades

loga(xy) = logax + logay loga(x/y) = logax − logay loga(xk) = k·logax

6️⃣ Logaritmo Decimal e Natural

✔ log x → base 10 ✔ ln x → base e

7️⃣ Exemplo Resolvido

log₂ 8 = ? 2³ = 8 → log₂ 8 = 3

8️⃣ Aplicações

✔ Escala Richter (terremotos) ✔ pH ✔ Intensidade sonora (decibéis) ✔ Crescimento populacional

9️⃣ Erros Comuns

❌ Esquecer que x > 0 ❌ Aplicar propriedades de forma incorreta ❌ Confundir base com argumento

🔟 Conexão

Base para equações logarítmicas e modelagem científica.
