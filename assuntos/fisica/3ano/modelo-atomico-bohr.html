🌟 Modelo Atômico de Bohr

3º Ano do Ensino Médio • Física • Professor: Arlisson Ferreira

🎯 Compreender como Bohr explicou os espectros atômicos usando a quantização da energia.

1️⃣ Problema do Modelo de Rutherford

Segundo a Física Clássica, o elétron em órbita deveria perder energia e colapsar no núcleo. Mas isso não ocorre.

2️⃣ Postulados de Bohr (1913)

✔ O elétron só pode ocupar órbitas específicas (órbitas estacionárias). ✔ Nessas órbitas, não há emissão de energia. ✔ A energia é emitida ou absorvida quando o elétron muda de nível.

3️⃣ Quantização do Momento Angular

mvr = n·h / 2π

n = 1, 2, 3...

4️⃣ Energia dos Níveis

Eₙ = −13,6 eV / n²

Válido para o átomo de hidrogênio.

5️⃣ Transições Eletrônicas

Quando o elétron salta de um nível para outro:

ΔE = hf

Se ΔE > 0 → absorção Se ΔE < 0 → emissão

6️⃣ Espectro de Balmer

Transições que terminam em n=2 produzem luz visível.

7️⃣ Exemplo Resolvido

Transição de n=3 para n=2.

E₃ = −13,6 / 9 = −1,51 eV E₂ = −13,6 / 4 = −3,4 eV

ΔE = −1,51 − (−3,4)

ΔE = 1,89 eV

8️⃣ Interpretação Física

A energia do átomo é quantizada. Não existem valores intermediários.

9️⃣ Limitações

✔ Funciona bem para hidrogênio ✔ Não explica átomos complexos

🔟 Importância Histórica

Foi a primeira aplicação concreta da quantização ao átomo.
