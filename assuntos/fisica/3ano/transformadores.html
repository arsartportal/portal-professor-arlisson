🎯 Compreender como a tensão elétrica pode ser elevada ou reduzida usando indução eletromagnética.

1️⃣ O que é um Transformador?

É um dispositivo que altera o valor da tensão elétrica alternada utilizando indução eletromagnética.

Ele funciona apenas com corrente alternada (CA).

2️⃣ Estrutura

✔ Bobina primária (Np) ✔ Bobina secundária (Ns) ✔ Núcleo de ferro

3️⃣ Relação Fundamental

Vs / Vp = Ns / Np

Onde:
✔ Vp = tensão primária ✔ Vs = tensão secundária ✔ Np = número de espiras primária ✔ Ns = número de espiras secundária

4️⃣ Transformador Elevador

Se Ns > Np → Vs > Vp Aumenta a tensão.

5️⃣ Transformador Abaixador

Se Ns < Np → Vs < Vp Reduz a tensão.

6️⃣ Conservação de Potência

Em um transformador ideal:

Pp = Ps

Vp · Ip = Vs · Is

7️⃣ Relação das Correntes

Ip / Is = Ns / Np

Se a tensão aumenta, a corrente diminui.

8️⃣ Aplicação no Sistema Elétrico

✔ Usinas elevam tensão (reduz perdas). ✔ Próximo às casas, transformadores abaixam tensão.

9️⃣ Exemplo Resolvido

Np = 200 espiras Ns = 1000 espiras Vp = 120 V

Vs = Vp · (Ns/Np)

Vs = 120 × (1000/200)

Vs = 600 V

🔟 Erros Comuns

❌ Achar que funciona com corrente contínua. ✔ Precisa de corrente alternada.
