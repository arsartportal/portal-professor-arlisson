🔥 Dilatação Térmica

2º Ano do Ensino Médio • Física • Professor: Arlisson Ferreira

🎯 Objetivo: Compreender como sólidos e líquidos variam suas dimensões quando submetidos a variações de temperatura.

📏 1. Conceito

Quando a temperatura de um corpo aumenta, suas partículas vibram com maior intensidade, fazendo com que o corpo aumente suas dimensões.

Esse fenômeno é chamado de dilatação térmica.

📌 O contrário ocorre quando há diminuição de temperatura (contração térmica).

📐 2. Dilatação Linear

Ocorre quando apenas uma dimensão é considerada (ex: comprimento).

ΔL = L₀ · α · ΔT

Onde:
✔ L₀ = comprimento inicial
✔ α = coeficiente de dilatação linear
✔ ΔT = variação de temperatura

⬛ 3. Dilatação Superficial

ΔA = A₀ · β · ΔT

Para sólidos:
β ≈ 2α

📦 4. Dilatação Volumétrica

ΔV = V₀ · γ · ΔT

Para sólidos:
γ ≈ 3α

🔢 5. Exemplo Resolvido

Uma barra de 2 m sofre aumento de temperatura de 50 °C. α = 2 × 10⁻⁵ °C⁻¹

ΔL = 2 · (2×10⁻⁵) · 50 ΔL = 0,002 m

Novo comprimento = 2,002 m

🔎 Glossário:

Dilatação • Coeficiente de Dilatação • Variação de Temperatura • Linear • Volumétrica
