Função Exponencial

Matemática – Ensino Médio

1. Definição

A função exponencial é aquela em que a variável aparece no expoente.
f(x) = ax
em que:

a é a base da potência, com:
a > 0 e a ≠ 1
Exemplos:

f(x) = 2x
f(x) = 3x
f(x) = (1/2)x
Ela aparece em várias situações do cotidiano, como crescimento populacional, juros compostos, reprodução de bactérias e decaimento radioativo.

2. Comportamento da função

O comportamento da função exponencial depende do valor da base.
Se a > 1, a função é crescente.
Se 0 < a < 1, a função é decrescente.
Além disso:

• o gráfico nunca toca o eixo x
• o domínio é o conjunto dos números reais
• a imagem é o conjunto dos números reais positivos
• toda função exponencial passa pelo ponto (0,1), porque:

a0 = 1

3. Exemplo resolvido

Considere a função:
f(x) = 2x
Calculando alguns valores:
x = -2 → f(x) = 2-2 = 1/4 = 0,25
x = -1 → f(x) = 2-1 = 1/2 = 0,5
x = 0 → f(x) = 20 = 1
x = 1 → f(x) = 21 = 2
x = 2 → f(x) = 22 = 4
Como a base é maior que 1, a função é crescente.

4. Aplicações no cotidiano

A função exponencial é muito usada para representar processos de crescimento rápido ou redução progressiva.
Crescimento populacional:
Uma população que dobra a cada período pode ser representada por:
P(t) = P₀ · 2t

Juros compostos:
Um capital aplicado com juros pode crescer segundo uma função exponencial.

Decaimento radioativo:
A quantidade de uma substância radioativa diminui exponencialmente com o tempo.
