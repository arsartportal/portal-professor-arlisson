1. Definição

A função quadrática, também chamada de função do 2º grau, é toda função escrita na forma:
f(x) = ax² + bx + c
em que:

a, b e c são números reais, com a ≠ 0.

Ela recebe esse nome porque o maior expoente da variável x é 2.
Exemplos:

f(x) = x² + 2x + 1
f(x) = 2x² - 3x + 5
f(x) = -x² + 4

2. Gráfico da função quadrática

O gráfico de uma função quadrática é uma parábola.

O valor de a determina a concavidade:
Se a > 0, a parábola tem concavidade voltada para cima.
Se a < 0, a parábola tem concavidade voltada para baixo.
Além disso:

• quanto maior o valor absoluto de a, mais “fechada” fica a parábola
• quanto menor o valor absoluto de a, mais “aberta” ela fica

3. Raízes da função quadrática

As raízes são os valores de x que fazem:
f(x) = 0
Para encontrá-las, usamos a fórmula de Bhaskara:
x = (-b ± √Δ) / 2a
onde:
Δ = b² - 4ac
O discriminante Δ indica a quantidade de raízes reais:
Se Δ > 0, existem duas raízes reais diferentes.
Se Δ = 0, existem duas raízes reais iguais.
Se Δ < 0, não existem raízes reais.

4. Vértice da parábola

O vértice é o ponto mais importante da parábola: ele representa o ponto de mínimo ou de máximo da função.
xv = -b / 2a
yv = -Δ / 4a
Se a concavidade é para cima, o vértice é um ponto de mínimo.
Se a concavidade é para baixo, o vértice é um ponto de máximo.

5. Exemplo resolvido

Considere:
f(x) = x² - 4x + 3
Temos:
a = 1, b = -4, c = 3
Discriminante:
Δ = (-4)² - 4·1·3 = 16 - 12 = 4

Raízes:
x = (4 ± √4)/2
x = (4 ± 2)/2
x₁ = 1 e x₂ = 3

Vértice:
xv = -(-4)/(2·1) = 2
yv = -4/(4·1) = -1

Vértice: (2, -1)
