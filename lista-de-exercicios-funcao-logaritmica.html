Função Logarítmica

Matemática – Ensino Médio

1. Definição

A função logarítmica é aquela em que a variável aparece dentro do logaritmo.
f(x) = loga(x)
em que:

a é a base do logaritmo, com:
a > 0 e a ≠ 1

Além disso, para o logaritmo existir, é necessário que:
x > 0
Exemplos:

f(x) = log2(x)
f(x) = log10(x)
f(x) = log1/2(x)

2. Ideia do logaritmo

O logaritmo responde à pergunta:
a? = x
Ou seja:
loga(x) = y ⇔ ay = x
log2(8) = 3, porque:
23 = 8

log10(100) = 2, porque:
102 = 100

3. Comportamento da função

O comportamento da função logarítmica depende da base.
Se a > 1, a função é crescente.
Se 0 < a < 1, a função é decrescente.
Além disso:

• o domínio é o conjunto dos números reais positivos
• a imagem é o conjunto dos números reais
• toda função logarítmica passa pelo ponto (1,0), porque:

loga(1) = 0

• a reta x = 0 é uma assíntota vertical

4. Relação com a função exponencial

A função logarítmica é a inversa da função exponencial.
Se:
y = ax

então a inversa é:
y = loga(x)
Isso significa que os gráficos dessas funções são simétricos em relação à reta y = x.

5. Exemplo resolvido

Considere a função:
f(x) = log2(x)
Calculando alguns valores:
x = 1 → f(x) = log2(1) = 0
x = 2 → f(x) = log2(2) = 1
x = 4 → f(x) = log2(4) = 2
x = 8 → f(x) = log2(8) = 3
x = 1/2 → f(x) = log2(1/2) = -1
Como a base é maior que 1, a função é crescente.
